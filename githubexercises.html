Exercise 1: Creating a repository

In this first exercise, we will create a repository. Repositories are used to store projects in GitHub and to master GitHub, you will need to be familiar with creating them. Follow along with this exercise to create your very first repository.

  1. On your GitHub homepage, locate the “New” button in the “Top repositories” column and click it.
  2. Write a name for your repository. This will be your personal repository, so get as creative as you want with the name.
  3. Add a description to your repository. For this exercise, add: “this is my first repository.”
  4. Visibility adjusts who can see your repository. Since it is a personal one solely for these exercises, click the dropdown arrow and use the “Private” option.
  5. Click the “Add README” button. README files are generally used as longer descriptions for the repository and explain what your project is about.
  6. Click the “Create repository” button. Don’t worry about the other options.

Congratulations, you've created your first repository!
